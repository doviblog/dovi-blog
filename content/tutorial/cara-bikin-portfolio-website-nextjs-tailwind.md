Portfolio website itu kartu nama digital kamu. Di tutorial ini aku bakal jelasin cara bikin dalam 1 hari.

## Setup

```bash
npx create-next-app@latest my-portfolio
cd my-portfolio
```

## Components to Build

1. **Navbar** - Navigation
2. **Hero** - Landing section
3. **About** - About me
4. **Projects** - Portfolio showcase
5. **Contact** - Contact form
6. **Footer** - Links & copyright

## Deploy

1. Push ke GitHub
2. Import ke Vercel
3. Add custom domain

Done! Portfolio live dalam 1 hari.

## Tips

1. Keep it simple
2. Fast loading
3. Mobile responsive
4. Show best work
5. Clear CTA
